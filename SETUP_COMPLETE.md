## 🔧 Customization Guide

### Change Colors
Edit `css/style.css` (lines 10-16):
```css
:root {
  --sv-cyan: #00e6fb;    /* Main cyan */
  --sv-blue: #1e88e5;    /* Blue accent */
  --sv-purple: #9c27b0;  /* Purple accent */
}
```

### Change Logo
Replace: `assets/favicon.png`

### Add/Edit Services
Edit: `services.html`
- Copy `.service-card` div
- Update icon, title, description

### Add/Edit Projects
Edit: `portfolio.html`
- Copy `.portfolio-card` div
- Update image, title, description

### Change Contact Info
