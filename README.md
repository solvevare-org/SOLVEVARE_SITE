# Solvevare Website - HTML/CSS/JavaScript Version

Complete conversion of the React/TypeScript Solvevare website to pure HTML, CSS, and JavaScript.

## 📁 Project Structure

```
SolvevareSite-HTML/
├── css/
│   └── style.css          # Main stylesheet with all styles
├── js/
│   └── script.js          # JavaScript for interactivity
├── images/                # Placeholder for images
├── index.html             # Home page
├── about.html             # About page
├── services.html          # Services page
├── portfolio.html         # Portfolio page
├── contact.html           # Contact page
└── README.md              # This file
```

## 📝 How to Use

1. **Setup Images**
   - Copy all images from `../SolvevareSite/public/` to the project root or update image paths
   - Update logo paths in HTML files
   - Update video paths in hero sections

2. **Open in Browser**
   - Simply open `index.html` in any modern web browser
   - No build process or dependencies required

3. **Customize**
   - Edit `css/style.css` for styling changes
   - Edit `js/script.js` for functionality changes
   - Update content directly in HTML files

## 🔧 Customization

### Colors
Edit CSS variables in `style.css`:
```css
:root {
  --sv-cyan: #00bcd4;
  --sv-blue: #1e88e5;
  --sv-purple: #9c27b0;
}
```

### Content
- Update text directly in HTML files
- Replace images in image tags
- Modify service/portfolio data in HTML

### Functionality
- Edit `js/script.js` for behavior changes
- Add new animations
- Modify form handling
